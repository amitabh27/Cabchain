# Cabchain
Decentralized implementation of Cab Service which involves peer-to-peer networking and blockchain. 

# Technology Stack

<ul>
  <li>Node JS</li>
  <li>React Native</li>
  <li>Mongo DB</li>
  <li>Android</li>
  <li>Blockchain Network</li>
</ul>
